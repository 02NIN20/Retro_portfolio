# Portafolio Personal - Lenin Coronel

Mi portafolio personal con estilo retro/terminal. Desarrollado con HTML, CSS y JavaScript vanilla.

## Características

- Diseño retro/terminal estilo Linux
- Tema claro/oscuro con toggle
- Animaciones y transiciones
- Totalmente responsive
- Sin dependencias externas (solo HTML, CSS, JS)

## Tecnologías

- HTML5
- CSS3 (Variables, Flexbox, Grid, Animaciones)
- JavaScript Vanilla

```
├── index.html

├── perfil.jpeg

```

## Customización

### Cambiar colors
Edita las variables en `styles.css`:
```css
:root {
  --green-console: #00ff00;
  --bg: #050805;
  --olive: #1e351e;
}
```

### Cambiar información
Edita el contenido en `index.html`:
- Nombre y descripción
- Proyectos
- Skills
- Links de contacto
